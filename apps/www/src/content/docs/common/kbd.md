## Keyboard Keys

List of supported keys.

- `command` - `⌘`
- `shift` - `⇧`
- `ctrl` - `⌃`
- `option` - `⌥`
- `enter` - `↵`
- `delete` - `⌫`
- `escape` - `⎋`
- `tab` - `⇥`
- `capslock` - `⇪`
- `up` - `↑`
- `right` - `→`
- `down` - `↓`
- `left` - `←`
- `pageup` - `⇞`
- `pagedown` - `⇟`
- `home` - `↖`
- `end` - `↘`
- `help` - `?`
- `space` - `␣`
